Jane Lite Wordpress Theme
---------------------

Jane Lite is a clean SuevaFree child theme with an optional slideshow on homepage, a new header layout in addition to the five header layouts available on SuevaFree and new Google Fonts. 
SuevaFree Essential Kit plugin is required to enable the one page section, available from WordPress.org

Created by ThemeinProgress, http://www.themeinprogress.com
