== Description ==
Temauno is a highly customizable theme based on Underscores with a responsive-design and a minimalist-style, which tries to be so quick and efficient as possible. Fonts, colors and social icons can be easily modified with the Customizer. It includes Spanish, German and French translations.

* Responsive Layout
* Custom Colors
* Custom Header

== Installation ==

1. In your admin panel, go to Appearance -> Themes and click the Add New button.
2. Click Upload and Choose File, then select the theme's ZIP file. Click Install Now.
3. Click Activate to use your new theme right away.

== Frequently Asked Questions ==

= What is the size of the thumbnails images showed on the index page? =
* 782px width and 160px height

= How do I change the colors? =

You can change the colors of your site easily using Temauno.

1. In your admin panel, go to Appearance -> Customize.
2. Now you will see the Customizer and a tab called 'Colors'. Click this tab.
3. You can now change the color of almost any element: text, links, background, menu, etc..
4. Once you are happy with your color changes you can click save and your changes will be reflected on your live site.

= How do I change the fonts? =

You can change the fonts of your site easily using Temauno.

1. In your admin panel, go to Appearance -> Customize.
2. Now you will see the Customizer and a tab called 'Fonts'. Click this tab.
3. You can now change fonts for the header and for the body
4. Once you are happy with your fonts changes you can click save and your changes will be reflected on your live site.

= How do I add the Social Links to the sidebar? =

Temeuno allows you display links to your social media profiles, like Twitter and Facebook, with icons.

1. In your admin panel, go to Appearance -> Customize.
2. Now you will see the Customizer and a tab called 'Social Icons'. Click this tab.
3. Add the your URL for every Icon you want to show.

= Can I publish some post or pages without Sidebar? =
Yes, you only have to select the "aside" post format for post. Pages are always without sidebar.
